# Battle of Memories

> — 記憶の欠片を集め、最強の陣を築け —

Auto Battler（オートバトラー）形式のブラウザゲームです。ショップでユニット・装備・魔法を購入し、全20ラウンドの自動戦闘を生き残ることを目指します。

---

## プレイ方法

1. `index.html` をブラウザで開く
2. タイトル画面から **Start Battle** を選択
3. 難易度（Easy / Normal / Hard）を選ぶ
4. ショップでユニットを購入 → 盤面に配置 → 「⚔️ 戦闘開始」

---

## ゲームの流れ

```
準備フェーズ → 戦闘フェーズ（自動）→ 結果 → 次のラウンド
```

- 全 **20ラウンド** 突破でクリア
- ❤ Life が **0** になるとゲームオーバー
- 敗北時は Life **-1**（ラウンド関係なし）
- 勝利時は Gold と EXP を獲得

---

## 主な機能

| 機能 | 概要 |
|---|---|
| ショップ | 毎ラウンド5枠にユニット・装備・魔法が出現 |
| 合成 | 同じユニット3体で自動合成（★アップ） |
| 装備 | ユニットに最大2個まで装着可能 |
| 魔法 | 準備フェーズに手動発動、持ち越し可 |
| ポーション | レベルアップ時に入手、ユニットを永続強化 |
| ミニイベント | R5・R10・R15 勝利後に宝箱イベント（3択） |
| 難易度 | Easy / Normal / Hard の3段階 |
| スコア記録 | 難易度別トップ5をローカルストレージに保存 |
| チュートリアル | 合成・装備・魔法を実際に体験しながら学べる |

---

## 技術仕様

- **実装** : 単一ファイル HTML（HTML / CSS / JavaScript）
- **外部依存** : なし（フォントのみ Google Fonts CDN）
- **対応環境** : モダンブラウザ（Chrome / Safari / Firefox）、スマートフォン対応
- **音声** : Web Audio API によるBGM・SE（合成音）
